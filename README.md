# картофель - ультимативное легковесное решение для создания картофеля

## установка картофеля
### нпм
```bash
$ npm i kartofel
```
### ярн
```bash
$ yarn add kartofel
```

## пример создания картофеля
### жабаскрипт
```js
const { создатьКартофель } = require("kartofel")

создатьКартофель()
```
### тайпскрипт
```ts
import { создатьКартофель } from "kartofel"

создатьКартофель()
```
